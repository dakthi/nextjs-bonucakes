KEY FEATURES
============

Cart Operations:
  • Add items to cart
  • Remove items from cart
  • Update quantities
  • Clear entire cart
  • Get cart count and totals

Promotions:
  • Buy 10 get 1 free (per product)
  • Automatic calculation
  • Multilingual messages

Shipping:
  • £8 flat rate
  • Applied when cart has items

Persistence:
  • localStorage integration
  • Data persists across sessions
  • Automatic migration

Multilingual:
  • English/Vietnamese support
  • Localized product names
  • Localized units and prices

Type Safety:
  • Full TypeScript coverage
  • Type-safe hooks
  • IntelliSense support


QUICK START
===========

1. Add to Navbar:
   import CartBadge from '@/components/CartBadge';
   <CartBadge />

2. Add to Product Page:
   import AddToCartButton from '@/components/AddToCartButton';
   <AddToCartButton product={cartProduct} />

3. Use Cart Data:
   import { useCartStore } from '@/lib/stores/cart-store';
   const items = useCartStore((state) => state.items);


STORE API
=========

Actions:
  addItem(product, quantity?)
  removeItem(productId)
  updateQuantity(productId, quantity)
  clearCart()

Getters:
  getCartCount()
  getSubtotal()
  getShippingFee()
  getTotal()
  getTotals()
  getPromotions()
  getCartForAPI()

Selector Hooks (optimized):
  useCartItems()
  useCartCount()
  useCartTotals()
  useCartPromotions()
